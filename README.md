# Facial Emotion Detection System

## Description
This project detects human facial emotions such as Happy, Sad, Angry, and Neutral 
using a machine learning model and displays the results on a web-based dashboard.

## Features
- Real-time facial emotion detection
- Emotion classification using machine learning
- Interactive dashboard for visualization
- Simple and user-friendly interface

## Technologies Used
- Python
- Machine Learning
- OpenCV
- HTML, CSS, JavaScript
- Node.js

## How to Run the Project

1. Clone the repository
   git clone https://github.com/Badrinath001/Emotion-detection
   
3. Navigate to the project folder
   cd minor project 2

4. Install dependencies
   pip install -r requirements.txt
   npm install

5. Run the application
   python server.py
